## Technology used
-   Python (back-end)
-   Kivy (front-end)
-   Bcrypt (database)
-   Mongodb (database)
-   figma (front-end)

## Try it yourself!
To set up, download the files and then install the required packages.  
Afterwards, create a file in the same directory named .env. Then put the following into the file:

```markdown
VAL=<MONGODB LINK>
```
Finally, run the main.py file.
